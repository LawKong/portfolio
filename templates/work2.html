002 : Image Processing App

This is a simple image processing application with GUI created by using Python 3.7. The chonky cat above is the sample image I used for this app.

Pillow and Tkinter are the main library. Pillow is the successor of PIL(Python Imaging Library which provides the image processing feature. Tkinter is the library that provides Python's standard GUI (Graphical User Interface).

Step-by-step guide to use this app

The initial appereance after launching the application with command "python3 images.py" at terminal.

A new window pops up after pressing the "Browse" button.

Look for the image file you want to edit.

The selected image name will be displayed. 4 process modes for you to choose.

Select resize to resize your image. Enter the width and height in the textboxes.

Press "Process" button to resize the image to the width and height you set. A new window pops up to show your resized image.
